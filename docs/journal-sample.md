## 🛠 RC탱크 프로젝트 — 첫 카메라 마운트 설계

오늘 드디어 rctank에 카메라 브라켓을 달기 시작했다. FreeCAD로 마운트 도면을 그리기 시작. 처음엔 카메라 각도를 고정할지 틸트를 줄지 고민했는데, 일단 고정 마운트로 먼저 검증하고 나중에 서보로 확장하기로 결론 냈다.

ROS2에서 image_transport로 영상 스트림 받는 예제 코드를 따라가 봤다. 생각보다 설정이 많지 않아서 내일 실제로 연결해볼 수 있을 것 같다.
- FreeCAD 마운트 도면 완성 후 출력 요청 예정
- ROS2 image_transport 패키지 설치 및 테스트 내일 진행

---

## 👥 인간관계 — 친구 J와 통화 / 여자친구와 저녁

대학 동기 J가 요즘 취업 준비 때문에 힘들다며 연락이 왔다. 30분 정도 통화했다. 나도 구직 중이라 해줄 수 있는 말이 많진 않았지만, 비슷한 처지라 그냥 들어주는 것 자체가 도움이 된 것 같았다. "같이 힘내자"는 말이 오갔다.

여자친구와는 저녁을 같이 먹었다. 요즘 서로 바빠서 오랜만에 여유롭게 앉아서 밥 먹는 시간이었다. 특별한 얘기보다 오늘 있었던 것들 나누는 것만으로도 충분했다. 지난번처럼 분위기 흐리지 않으려고 즉흥 계획은 미리 얘기했다.

## 💼 구직 — Java Spring 스터디 + 이력서 보완 + 코테

오전에 Java Spring MVC 흐름 정리. DispatcherServlet → HandlerMapping → Controller → ViewResolver 순서를 그림으로 직접 그려보니 확실히 머릿속에 들어왔다. 암기가 아니라 구조를 이해하는 방식으로 접근하니 훨씬 낫다.

오후에 이력서 프로젝트 설명 부분을 다듬었다. RC탱크 토이 프로젝트를 "임베디드 + ROS2 + 컴퓨터비전 연동 경험"으로 포지셔닝하는 방향으로 잡았다. 아직 완성은 아니지만 방향이 잡혔다.

코딩 테스트 문제 2개. 그래프 BFS 문제인데 한 개는 맞고 한 개는 틀렸다. 틀린 건 방문 처리 위치 실수였다. 큐에 넣을 때 바로 visited 처리하는 습관을 들여야겠다.
